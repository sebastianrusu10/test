Domain 1 – Operations Using Data Types and Operators

1.1 Core Data Types & Type Conversion

Identify built-in types, select the correct type for a scenario, and convert types when combining input, calculations, and output.

Scalar types

  • int – whole numbers: 5, -3, 100
  • float – decimal / floating-point: 3.14, 2.0, 1e3
  • str – text: "hello", "42", 'A'
  • bool – logical values: True, False

Collection types

  • list – ordered, mutable: [1, 2, 3]
  • tuple – ordered, immutable: (1, 2, 3)
  • dict – key–value mapping: {"gold": 100}
  • set – unique elements, no fixed order: {1, 2, 3}

Type checking and conversion

Exam items often show a snippet and ask for the type of a variable or the conversion needed to fix an error.


# check type
type(42)                 # <class 'int'>
isinstance(42, int)      # True

# numeric & string conversion
int("42")                # 42
float(15)                # 15.0
str(10)                  # "10"

# boolean conversion: 0 / empty -> False, non-zero / non-empty -> True
bool(0)                  # False
bool(10)                 # True
bool("")                 # False
bool("hi")               # True

# build collections from other types
list("abc")              # ['a', 'b', 'c']
tuple([1, 2])            # (1, 2)
set([1, 1, 2])           # {1, 2}

int() conversion details and common errors


# valid decimal strings
int("10")         # 10
int("+10")        # 10
int("-10")        # -10

# invalid (ValueError)
# int("10.5")     # ValueError: invalid literal
# int("abc")      # ValueError

# optional base
int("1010", 2)    # 10 (binary)
int("ff", 16)     # 255 (hex)
int("20", 8)      # 16 (octal)

# typical exam trap
value = "007"
print(int(value)) # 7 (leading zeros ignored)

1.2 Indexing, Slicing & List Operations

Access individual elements and slices; add, insert, delete, sort, merge, and reverse lists.

Indexing & slicing (strings and sequences)


serial_number = "1111-2222-3333-4444"

# indexing (0-based)
serial_number[0]         # '1'
serial_number[5]         # '2'
serial_number[-1]        # '4' (last character)

# slicing: seq[start:stop] (stop not included)
serial_number[-4:]       # '4444'
serial_number[5:14]      # '2222-3333'
serial_number[:4]        # '1111'
serial_number[::2]       # every second character

Lists – create, modify, remove


capitals = ["Chisinau", "Bucharest", "Kiev"]

# add / insert / replace
capitals.append("Moscow")           # add at end
capitals.insert(1, "Iasi")          # insert at index 1
capitals[0] = "Cluj"                # replace item at index 0

# remove elements
capitals.remove("Kiev")             # remove first matching value

last_item = capitals.pop()          # remove and return last item
second_item = capitals.pop(1)       # remove item at index 1

Sorting, merging, reversing


# sorting
capitals = ["Tokyo", "Berlin", "London"]
capitals.sort()                     # ['Berlin', 'London', 'Tokyo']
capitals.sort(reverse=True)         # ['Tokyo', 'London', 'Berlin']

# merging lists
a = [1, 2]
b = [3, 4]

c = a + b           # [1, 2, 3, 4] (creates new list)
a.extend(b)         # a becomes [1, 2, 3, 4]

# reversing
nums = [1, 2, 3, 4]
nums.reverse()      # [4, 3, 2, 1] (in-place)

# useful built-ins
population = [100, 101, 102]
max(population)     # 102
min(population)     # 100
len(population)     # 3
sum(population)     # 303

1.4 Operators, %, //, Unary + / - and Precedence

Evaluate expressions combining arithmetic, comparison, and logical operators.

Arithmetic & special operators % and //


# basic arithmetic
5 + 2      # 7
5 - 2      # 3
5 * 2      # 10
5 / 2      # 2.5      (true division)

# floor division and modulus
5 // 2     # 2        (floor division)
5 % 2      # 1        (remainder)

# relationship between // and %
a = 5
b = 2
q = a // b          # quotient
r = a % b           # remainder
# a == b * q + r    # always holds for integers

# negative numbers: // is floor, not truncate
-5 // 2    # -3  (floor: next lower integer)
-5 % 2     # 1   (because -5 = 2 * (-3) + 1)

Comparison, logical, membership, identity


x = 5

# comparison
x == 5     # True
x != 5     # False
x > 3      # True
x <= 7     # True

# logical
age = 20
has_id = True

(age >= 18) and has_id      # True
(age < 18) or has_id        # True
not has_id                  # False

# membership
colors = ["red", "green", "blue"]
"red" in colors             # True
"yellow" not in colors      # True

# identity vs equality
a = [1, 2, 3]
b = a
c = [1, 2, 3]

a is b      # True  (same object in memory)
a is c      # False (different objects)
a == c      # True  (same contents)

Unary + and -


y = 5
-y     # -5
+y     # 5   (returns value with same sign)

Operator precedence (exam level)

  1. ** (exponent)
  2. Unary +, -, not
  3. *, /, //, %
  4. +, -
  5. Comparisons (< <= > >= == !=)
  6. and
  7. or

result = 2 + 3 * 4 > 10 and not False
# 3 * 4 = 12
# 2 + 12 = 14
# 14 > 10 -> True
# not False -> True
# True and True -> True

# break: exit loop immediately
for i in range(1, 10):
    if i == 5:
        break
    print(i)          # 1 2 3 4

# continue: skip to next iteration
for i in range(1, 6):
    if i % 2 == 0:
        continue
    print(i)          # 1 3 5

# pass: placeholder (does nothing)

    if i % 2 == 0:
        pass          # TODO: handle even numbers later
    else:
        print(i)      # 1 3

Index vs element when looping

# indexing with range(len())

    print("index", i, "value", values[i])
# index 0 value 10
# index 1 value 20

Common infinite-loop trap


x = 0
while x < 3:
    print(x)
    # x += 1   # missing increment => infinite loop

# correct

while x < 3:
    print(x)
    x += 1
# 0
# 1
# 2

Common output-question traps

  • range(n) is 0..n-1, range(a, b) never includes b.
  • Nested loops: total iterations = product of ranges if no break.
  • break stops only the innermost loop, not outer loops.
  • continue skips the rest of the body for that iteration only.
  • for-else / while-else run else only when the loop ends without break.

Domain 3 – Input and Output Operations

3.1 File Handling: open, modes, with

Choose correct file modes and predict behavior when reading and writing.

Opening files

Pattern: open(filename, mode, encoding="utf-8") for text files.

  • "r" – read (default), error if file does not exist.
  • "w" – write, truncate file or create new.
  • "a" – append, create if missing, writes go to end.
  • "x" – create, fail if file exists.
  • "rb", "wb", "ab" – binary modes (no text encoding).
  • "r+" – read and write, file must exist, no truncate.
  • "w+" – read and write, truncate or create new.
  • "a+" – read and append, file pointer at end for writes.

message = open("1.txt", "r", encoding="utf-8")
content = message.read()
message.close()

with statement (context manager)


with open("1.txt", "r", encoding="utf-8") as message:
    content = message.read()
    print(content)
# file automatically closed here

Reading and writing


# read all text as a single string
with open("1.txt", "r", encoding="utf-8") as f:
    content = f.read()

# read one line at a time
with open("1.txt", "r", encoding="utf-8") as f:
    first_line = f.readline()
    second_line = f.readline()

# read all lines into list
with open("1.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()    # ['line1\\n', 'line2\\n', ...]

# iterate directly over file object (line-by-line)
with open("1.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())

Write / append / read-write modes


# write (overwrites file)
with open("log.txt", "w", encoding="utf-8") as f:
    f.write("Start log\\n")
    f.write("Line 2\\n")

# append (adds at end, keeps old content)
with open("log.txt", "a", encoding="utf-8") as f:
    f.write("Appended line\\n")

# read and write without truncating (r+)
with open("data.txt", "r+", encoding="utf-8") as f:
    text = f.read()
    f.seek(0, 2)           # move to end of file
    f.write("\\nMore data")

# w+ truncates file first
with open("temp.txt", "w+", encoding="utf-8") as f:
    f.write("Hello")
    f.seek(0)
    print(f.read())        # 'Hello'

Common file patterns (exam-style)


# copy all lines from input.txt to output.txt
with open("input.txt", "r", encoding="utf-8") as src:
    with open("output.txt", "w", encoding="utf-8") as dst:
        for line in src:
            dst.write(line)

# count lines in a file
count = 0
with open("data.txt", "r", encoding="utf-8") as f:
    for _ in f:
        count += 1
print("Lines:", count)

# append a line only if file exists
import os

if os.path.exists("log.txt"):
    with open("log.txt", "a", encoding="utf-8") as f:
        f.write("Another entry\\n")
else:
    print("log.txt not found")

Checking existence, size, and basic os utilities


import os
import os.path

path = "data"
file_name = "scores.txt"
full_path = os.path.join(path, file_name)

# ensure directory exists
if not os.path.isdir(path):
    os.makedirs(path)

# write a file in that directory
with open(full_path, "w", encoding="utf-8") as f:
    f.write("10\\n20\\n30\\n")

# check existence and size
if os.path.exists(full_path):
    size = os.path.getsize(full_path)
    print("File size:", size, "bytes")

# rename file
new_path = os.path.join(path, "scores_old.txt")
os.rename(full_path, new_path)

# remove file

    os.remove(new_path)

def func(a, b, c=10):
    print(a, b, c)

5.3 Unit Testing with unittest

Unit tests verify that functions and methods behave as expected.


import unittest

def double(x):
    return x * 2

class TestMain(unittest.TestCase):

    def test_double_positive(self):      # method name must start with test_
        self.assertEqual(double(3), 6)

    def test_double_zero(self):
        self.assertEqual(double(0), 0)

if __name__ == "__main__":
    unittest.main()

Common assertion methods (exam list)


self.assertEqual(a, b)         # a == b
self.assertNotEqual(a, b)      # a != b
self.assertTrue(x)             # bool(x) is True
self.assertFalse(x)            # bool(x) is False
self.assertIn(a, b)            # a in b
self.assertIs(a, b)            # a is b (same object)
self.assertIsInstance(obj, cls)    # isinstance(obj, cls)

assertIsInstance example


def square(x):

        self.assertIsInstance(result, int)
        self.assertEqual(result, 9)

More unittest assertion methods


import unittest

class TestValues(unittest.TestCase):

    def test_none(self):
        value = None
        self.assertIsNone(value)
        # self.assertIsNotNone(value)  # would fail

    def test_membership(self):
        data = [1, 2, 3]
        self.assertIn(2, data)
        self.assertNotIn(4, data)

    def test_float_almost_equal(self):
        x = 0.1 + 0.2
        self.assertAlmostEqual(x, 0.3, places=7)

    def test_exception(self):
        with self.assertRaises(ZeroDivisionError):
            1 / 0

Common unittest output patterns

Common date-only strftime formats


import datetime

d = datetime.date(2025, 11, 25)  # 25 November 2025

# numeric variants (month/day/year and day/month/year)
print(d.strftime("%m-%d-%Y"))  # 11-25-2025
print(d.strftime("%d-%m-%Y"))  # 25-11-2025
print(d.strftime("%Y-%m-%d"))  # 2025-11-25

print(d.strftime("%m/%d/%Y"))  # 11/25/2025
print(d.strftime("%d/%m/%Y"))  # 25/11/2025
print(d.strftime("%Y/%m/%d"))  # 2025/11/25

print(d.strftime("%m.%d.%Y"))  # 11.25.2025
print(d.strftime("%d.%m.%Y"))  # 25.11.2025

# with 2-digit year
print(d.strftime("%m-%d-%y"))  # 11-25-25
print(d.strftime("%d-%m-%y"))  # 25-11-25

# textual month names
print(d.strftime("%B %d, %Y"))   # November 25, 2025
print(d.strftime("%d %B %Y"))    # 25 November 2025
print(d.strftime("%b %d, %Y"))   # Nov 25, 2025
print(d.strftime("%d %b %Y"))    # 25 Nov 2025

# weekday + date
print(d.strftime("%A, %B %d, %Y"))   # Tuesday, November 25, 2025
print(d.strftime("%a, %d %b %Y"))    # Tue, 25 Nov 2025

# ISO-like forms
print(d.strftime("%Y-%m-%d"))        # 2025-11-25  (ISO date)
print(d.strftime("%G-W%V-%u"))       # ISO year-week-weekday

Common date+time strftime formats


import datetime

dt = datetime.datetime(2025, 11, 25, 21, 5, 3)

# 24-hour clock
print(dt.strftime("%Y-%m-%d %H:%M"))       # 2025-11-25 21:05
print(dt.strftime("%d/%m/%Y %H:%M:%S"))    # 25/11/2025 21:05:03

# 12-hour clock with AM/PM
print(dt.strftime("%m-%d-%Y %I:%M %p"))    # 11-25-2025 09:05 PM
print(dt.strftime("%I:%M:%S %p on %A"))    # 09:05:03 PM on Tuesday

# with microseconds
dt2 = datetime.datetime(2025, 11, 25, 21, 5, 3, 123456)
print(dt2.strftime("%Y-%m-%d %H:%M:%S.%f"))
# 2025-11-25 21:05:03.123456

# compact numeric formats
print(dt.strftime("%Y%m%d"))           # 20251125
print(dt.strftime("%m%d%Y"))           # 11252025

import random

random.random()             # float in [0.0, 1.0)
random.randint(1, 6)        # int in [1, 6] (both ends inclusive)
random.randrange(5, 20, 2)  # odd number 5..19 (stop excluded)

categories = ["Action", "Comedy", "Drama", "Horror"]

random.shuffle(categories)         # shuffle list in place
choice = random.choice(categories) # one random element
two_random = random.sample(categories, 2)  # 2 distinct elements

# randint vs randrange (boundaries)
print(random.randint(1, 3))       # 1, 2 or 3
print(random.randrange(1, 4))     # 1, 2 or 3
